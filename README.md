**TITAN** is a high-performance investigation framework designed for massive data correlation. It enables investigators to map complex influence networks, trace opaque financial flows, and reconstruct deleted digital footprints using open-source intelligence.

---

## 🛠️ Core Features

* **Variable Intelligence System:** TITAN’s core strength. An integrated, easy-to-use system to manage, track, and correlate variables (IDs, handles, emails) across your entire investigation.
* **Smart Website Scraper:** Automatically pulls essential branding data including Website Names and Logos to populate your workspace instantly.
* **Local Link Analysis:** A built-in graph engine to visualize connections between your variables directly on your machine.
* **Privacy-First:** No cloud uploads. All your variables, scraped data, and investigation maps stay 100% local.
* **Modern Interface:** A clean, high-end UI designed to make complex variable management feel intuitive and visual.

## 🚀 Installation & Setup

### Prerequisites
* **Python 3.10+**
* **requirements.txt**

### Installation
1. **Clone the repository:**
   ```
   git clone https://github.com/med0-0/Midins-Titan---Osint-Tool
   cd Midins Titan V1.1.0
Install dependencies:
```
pip install -r requirements.txt
```
**Launch the system:**

```
python app.py
```
(Or launch.bat for windows)
